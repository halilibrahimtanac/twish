# Twish

Next.js 15 app with TRPC, Drizzle, and WebRTC for micro-blogging plus calls. Create short posts, comment/quote/retwish, upload media, and hop on calls with other users.

## Live
- https://twish-t5jb.vercel.app

## Features
- Email authentication and editable profile cards
- Create twishes, comment, retwish/quote, like, and preview media
- User search, feed, and data fetching via TRPC
- WebRTC + Socket-powered call modal and real-time notifications
- Cloudinary uploads plus Drizzle ORM and NextAuth integrations

## Local Setup
1) Clone the repo and install deps: `npm install`
2) Start dev server: `npm run dev` and open `http://localhost:3000`

## Scripts
- `npm run dev` — run in dev mode
- `npm run build` — build for production
- `npm start` — start the built app
- `npm run lint` — static analysis
